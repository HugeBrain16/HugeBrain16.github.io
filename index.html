Simple command loader for cmdtools framework

Author: HugeBrain16
Date: 29 Aug 2022
Category: Guide
Keywords: cmdtools, framework, command, loader, python3

cmdtools repository

Scuffed guide for loading multiple command modules by using this ((simple)) loader.
We have this directory structure for example: 

./ <- root dir
    commands/
        basic.py
        advanced.py
    main.py
and you want to load all commands in commands directory.
each modules must follow this kind of structure: 
                
...
from cmdtools.ext.command import Group

group = Group("foobar")

"""commands to append to group container..."""
...
in main.py file, define this function for example: 
                
import os
import importlib
from typing import List
from cmdtools.ext.command import Group
# lets assume you've imported all of this

def load_commands(commands_dir: str) -> List[Group]:
    commands = []

    for file in os.listdir(commands_dir):
        if file.endswith(".py"):
            modname = file.rsplit(".py", 1)[0] # strip file extension
            modpath = os.path.join(commands_dir, modname).replace(os.sep, ".") # make importable dir
            mod = importlib.import_module(modpath)

            if hasattr(mod, "group"):
                if isinstance(mod.group, Group):
                    commands.append(mod.group)

    return commands
